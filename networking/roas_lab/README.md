#Network ROAS Configuration 
- Engineered a secure Router-on-a-Stick (ROAS) network topology with 5 VLANs, 802.1Q trunking, and inter-VLAN routing utilizing Cisco Packet Tracer. 
- Configured SSH authentication for secure remote management and used VLSM for efficient subnetting, supporting 296 host devices. 
- Deployed the design on live Cisco Enterprise equipment, demonstrating troubleshooting and hands-on networking skills
