# Home Networking & Virtualization Lab
#### Cisco ISR 1111 | Proxmox VE | Tailscale | VLAN Segmentation | NAT | DHCP | Virtualized Services
- Designed and deployed an enterprise-style home network to practice routing, segmentation, and virtualization concepts.
- Configured Cisco ISR 1111 as the core router with VLANs, static routing, NAT overload, DHCP, LLDP, and ACLs.
- Deployed Proxmox VE to host virtualized services and configured Tailscale for secure remote access to the lab environment.
